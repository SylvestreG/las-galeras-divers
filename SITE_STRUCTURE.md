## 🎯 Interactions Utilisateur

```
HOVER EFFECTS
├── Cards → Lift + Shadow
├── Buttons → Color Change + Lift
├── Images → Scale + Overlay
├── Links → Color Change
└── Gallery → Overlay + Icon

CLICK ACTIONS
├── Nav Links → Smooth Scroll
├── CTA Buttons → Navigate to Contact
├── Gallery Images → Open Lightbox
├── Social Links → Open in New Tab
├── Phone/Email → Open Default App
└── Mobile Menu → Toggle Navigation

ANIMATIONS
├── Fade In → On Scroll (sections)
├── Slide In → Service Cards
├── Bounce → Scroll Indicator
├── Scale → Hover Effects
└── Smooth → All Transitions
```
